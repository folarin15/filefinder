# Physiology Word Practical Finder

Static GitHub Pages-ready page for students to find their physiology word practical files by matric number.

## How to use

1. Keep the `.docx` files inside `files/Practical 1` and `files/Practical 2`.
2. Push this folder to a GitHub repository.
3. Enable GitHub Pages for the repository.

## Note

GitHub Pages is public hosting. This is fast and simple, but not private file storage.
